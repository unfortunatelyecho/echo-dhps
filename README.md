# Installation Instructions:
Download & Place the folder in your resources directory
Rename the folder to heist-system or your preferred name
Configure config.lua:


- Set `Config.Framework to 'qbcore', 'qbx', or 'esx' (or 'auto' for auto-detection)`
- Adjust settings to your preference
- Add to server.cfg:
- `ensure echo-dhps`
- Dependencies:
- oxmysql
- ox_lib (recommended)
- Your framework (QBCore/QBX/ESX)
- qb-target or ox_target (if using target system)
- Restart your server
